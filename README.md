# lyuweiyi-MambaMDN
This is the official implementation of MambaMDN, a dual-domain framework designed for high-fidelity multi-contrast MRI reconstruction.
<img width="3552" height="1416" alt="Model" src="https://github.com/user-attachments/assets/2ac1a2ca-13ac-4d11-a219-e7800a751160" />

The repository is established for project tracking. The full source code (including training/evaluation pipelines, data processing scripts, and pre-trained models) will be publicly released upon formal acceptance of the paper.
Key Contributions
Dual-domain framework for multi-contrast MRI reconstruction
Mamba-based modality disentanglement module to eliminate cross-contrast interference
Superior performance (up to 1.15 dB/0.26 dB PSNR gain on IXI/BraTS datasets)
